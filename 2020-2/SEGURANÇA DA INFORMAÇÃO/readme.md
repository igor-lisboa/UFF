# Cifras de Stream vs Cifras de Bloco
* Diferença:
    * Cifra de Stream
        * Mensagem processada Bit a Bit
        * chegam em fila
        * sao processados individualmente
    * cifra de blocos
        * bits processados em paralelo
        * valores na ordem de centenas de bits
